- `styles.css`
- `script.js`
- `photos/photo1.JPG`
- `photos/photo2.JPG`
- `photos/photo3.JPG`
- `photos/photo4.jpg`
- `photos/photo5.jpg`

## Quick edits
1. Update text in `index.html` and `script.js`.
2. Replace photos in `photos/` using the same filenames.
3. Commit and push to `main`.

## Deploy
GitHub Pages is configured from:
- Branch: `main`
- Folder: `/` (root)

Any push to `main` auto-deploys the site.
